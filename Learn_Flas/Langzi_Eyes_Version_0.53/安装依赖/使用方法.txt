1. 必备环境
	【python2】
	【mysql】
2. 依赖包

【pip install -r requirement.txt】
如果安装失败依次使用pip install xxx 安装依赖包


然后编辑 app/web/config.py文件

设置好数据库名，账号密码，端口号

登陆账号密码


启动方式：在app目录下，cmd下输入

【python run.py】

然后浏览器打开

http://127.0.0.1:5000/
